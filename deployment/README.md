# 📦 חבילת פריסה - מערכת תאוריה דיגיטלית

חבילה להתקנת קיצורי דרך למערכת מבחני התאוריה במחשבי כיתה.

## 📁 קבצים בחבילה

| קובץ | תפקיד |
|---|---|
| `Install-ExamShortcuts.ps1` | סקריפט PowerShell ראשי - עושה את כל העבודה |
| `install-teacher.bat` | הפעלה במחשב מורה (4 קיצורים: בוחן, מורה, נבחן, תרגול) |
| `install-student.bat` | הפעלה במחשב תלמיד (2 קיצורים: נבחן, תרגול) |
| `README.md` | המסמך הזה |

## 🎯 מה נוצר על המחשב

הסקריפט יוצר **קיצורי דרך בשולחן העבודה** שפותחים את הדפים בחלון נקי של הדפדפן (מצב אפליקציה — בלי סרגל כתובות וכרטיסיות). כל קיצור מגיע עם האייקון הייעודי שלו (לוגו רישוי צה"ל בצבע המתאים).

### במחשב מורה

| קיצור | צבע | דף |
|---|---|---|
| 🔵 בוחן | כחול | `examiner.html` |
| 🟣 מורה | סגול | `teacher.html` |
| 🟢 נבחן | ירוק | `examinee.html` |
| 🟤 תרגול | זהב | `student.html` |

### במחשב תלמיד

| קיצור | צבע | דף |
|---|---|---|
| 🟢 נבחן | ירוק | `examinee.html` |
| 🟤 תרגול | זהב | `student.html` |

## 🖥️ איפה הקיצורים נשמרים

- **אם הסקריפט רץ כמנהל (מומלץ):** ב-`C:\Users\Public\Desktop` — כל משתמש שמתחבר למחשב יראה את הקיצורים.
- **אם לא כמנהל:** בשולחן העבודה של המשתמש הנוכחי בלבד.

האייקונים נשמרים ב:
- אדמין: `C:\ProgramData\ExamShortcuts\icons\`
- משתמש רגיל: `%LOCALAPPDATA%\ExamShortcuts\icons\`

## ▶️ שימוש ידני (מחשב בודד)

1. העתק את תיקיית `deployment` למחשב היעד (מקל USB, תיקייה משותפת, וכו').
2. **לחץ לחיצה ימנית** על `install-teacher.bat` או `install-student.bat`.
3. בחר **"Run as administrator"** (מומלץ — כדי שהקיצורים יהיו זמינים לכל המשתמשים).
4. המתן לסיום הסקריפט.
5. הקיצורים יופיעו בשולחן העבודה.

## 🌐 דרישות

- Windows 10 או מעלה
- PowerShell 5.1 (מובנה ב-Windows)
- **חיבור לאינטרנט** — הסקריפט מוריד את האייקונים מ-GitHub Pages בהפעלה הראשונה
- דפדפן Edge או Chrome (אופציונלי, אבל מומלץ מאוד — נותן חוויית "אפליקציה" נקייה)

## 🔄 עדכון אייקונים בעתיד

אם תעדכן את האייקונים ב-GitHub, פשוט הרץ את הסקריפט שוב — הוא מוריד מחדש בכל פעם. הקיצורים הישנים יוחלפו אוטומטית.

## 🔧 פרמטרים מתקדמים

אפשר להריץ את סקריפט ה-PowerShell ישירות עם פרמטרים:

```powershell
# התקנה רגילה
.\Install-ExamShortcuts.ps1 -Role Teacher
.\Install-ExamShortcuts.ps1 -Role Student

# שינוי כתובת בסיס (למשל אם האתר עובר לדומיין אחר)
.\Install-ExamShortcuts.ps1 -Role Student -BaseUrl 'https://example.com/exam'
```

## 🚀 פריסה המונית (בהמשך)

תסריט פריסה המונית (`Deploy-Classroom.ps1`) שיפיץ את החבילה מהמחשב של המורה ל-30 מחשבי הכיתה יתווסף לאחר שנברר:

1. האם המחשבים בדומיין (Active Directory) או workgroup
2. האם WinRM מופעל
3. האם יש הרשאות אדמין משותפות

בהתאם לכך נבחר בין אחת מהגישות הבאות:

| שיטה | דרישות | יתרון |
|---|---|---|
| **PowerShell Remoting** (`Invoke-Command`) | דומיין + WinRM + אדמין | הכי מהיר — שורה אחת מפיצה ל-30 מחשבים |
| **Group Policy (GPO)** | דומיין + Group Policy | הכי מסודר — עובד אוטומטית גם על מחשבים חדשים |
| **PsExec** | כל אדמין ברשת | עובד גם ב-workgroup, לא דורש WinRM |
| **תיקייה משותפת** | תיקייה ברשת | אפס דרישות, תלמיד מריץ ידנית פעם אחת |

## ❓ פתרון בעיות

### הקיצורים נוצרו אבל לא פותחים דף
- ודא שיש חיבור לאינטרנט במחשב
- ודא ש-Edge או Chrome מותקנים

### שגיאה "execution policy"
- ה-`.bat` כבר מטפל בזה אוטומטית (עם `-ExecutionPolicy Bypass`)
- אם אתה מריץ את ה-`.ps1` ישירות: הפעל PowerShell כמנהל וכתוב `Set-ExecutionPolicy -Scope Process Bypass`

### "לא זוהה Edge/Chrome"
הסקריפט עדיין ייצור קיצורים, אבל הם יפתחו את הדפים בדפדפן ברירת המחדל כחלון רגיל (לא במצב אפליקציה). זה עובד, אבל ללא החוויה הנקייה.

### האייקון לא מופיע נכון
- נסה לנעול מחדש (`Lock` + `Unlock`) או הפעל מחדש את explorer.exe
- ודא שהאייקונים קיימים ב-`C:\ProgramData\ExamShortcuts\icons\`
